# PIC Soft Start

An 230VAC 2kVA Soft start circuit with PIC16F15313 microcontroler.

# Specification

* Input voltage: 230VAC
* Input frequency: 50Hz
* Max power output: 2kVA
* Phase angle control: 0° - 180°
* Slope time: 0.1sec - 10sec
